getAvailableWidgetCompilers()

Source:
  • project.js, line 100
Returns:

Dictionary of available widget compilers. The key is the widget name, the value is an object:

  • path: absolute path of the compiler's' directory.
  • name: widget's name.
  • compiler: compiler's module owning functions such as compile, precompile, ...
  • precompilation: is this widget in mode precompilation? In this case, it must be called in the Top-Down walking.
