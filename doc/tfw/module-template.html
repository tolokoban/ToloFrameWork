Module: template

template

A template is a text with directives. A directive is enclosed in double-curlies. Here is an example :

Welcome mister {{name}}! I'm happy to see you again.

Source:
  • template.js, line 1

Methods

<static> text(text, replacer)

Parameters:
Name Type Description
text NAME

Template text with $ directives.
replacer

Can be of two types:

  • object: A map between the directives' names and the value of replacement.
  • function: A function with the directive's name as unique argument. It must return the replacement string. this is used as a context along all replacements. You can add any add attributs in it, but please avoid reserved one: text, count, out, cursor. The last gives the cursor's position when the directive has been reached.
Source:
  • template.js, line 62
Returns:

The context object with at least theses attributes :

  • text: Initial text.
  • count: Number of replacements made.
  • out: Text after replacements.
Example
var Tpl = require("./template");

var text = "Hi {{Joe}} ! Who's {{Natacha}} ?";
console.log(text);
var replacer = function(name) {
    console.log("name: \"" + name + "\"");
    return "<data>" + name + "</data>";
};

var ctx = Tpl.text(text, replacer);
console.log(ctx.out);

ctx = Tpl.text(
    text, 
    {
        joe: "buddy",
        natacha: "that girl"
    }
);
console.log(ctx.out);
